# /zonaprop:stats

Mostrar estadísticas del mercado inmobiliario de la base de datos.

## Instrucciones

1. Ejecutar `zonaprop_db_stats` para obtener el resumen.
2. Presentar los datos de forma clara: cantidad por barrio, promedios de precio/m², fecha del último escaneo.
3. Destacar los barrios más caros y más baratos.
4. Si el usuario pregunta por un barrio específico, dar el detalle de ese barrio.
