# /zonaprop:oportunidades

Mostrar las mejores oportunidades de inversión inmobiliaria.

## Instrucciones

1. Ejecutar `zonaprop_db_stats` para tener contexto del mercado.
2. Ejecutar `zonaprop_oportunidades` con limit=10.
3. De las top 3, ejecutar `zonaprop_analizar_db` con fotos para análisis visual.
4. Presentar un resumen comparativo con recomendación final.
5. Alertar si alguna parece tener m² inflados (publicación fraudulenta).
