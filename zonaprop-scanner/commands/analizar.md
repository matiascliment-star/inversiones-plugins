# /zonaprop:analizar

Analizar visualmente propiedades de un barrio específico.

## Instrucciones

1. Preguntar al usuario qué barrio o filtros quiere (si no los especificó).
2. Ejecutar `zonaprop_analizar_db` con los filtros indicados y fotos=4.
3. Para cada propiedad, dar un análisis visual detallado:
   - Estado general, terminaciones, luminosidad
   - Red flags visibles en las fotos
   - Potencial de refacción
   - Puntaje 1-10 como inversión
4. Comparar entre las propiedades y recomendar la mejor opción.
